Ik ben Joren. In mijn vrije tijd doe ik al een tijdje aan programmeren, maar sinds dit jaar studeer ik het ook in het graduaat Programmeren bij Thomas More.
Ik werk al een 6-tal jaar met Java, maar ik leer toch steeds dingen bij. Het is al lang geleden dat ik zelf nog html heb gebruikt, ik edit vaker
bestaande paginas (zowel de html code zelf als kleine aanpassingen via userscripts).

Ik heb eerst twee jaar van mijn middelbaar STEM gestudeerd bij Sint-Gabriël in Boechout.
Hierna heb ik 3 jaar Industriële Wetenschappen gestudeerd bij GTI in Mortsel.
Uiteindelijk ben ik door de achterstand die ik opliep vanwege Covid-17 voor mijn laatste jaar Toegepaste ICT gaan studeren bij Don Bosco in Hoboken.

Je kan mij elke vrijdag en zaterdag avond vinden in het jeugdhuis Tjok in Hove. Daar ben ik wekelijks aan het tappen,
bonnen aan het verkopen bij evenementen, op een andere manier aan het helpen of gewoon gezellig aan het praten met vrienden.
Ook beheer ik momenteel de website van het jeugdhuis als co-verantwoordelijke voor sociale media.

Voor meer over mij kan je altijd mijn CV bekijken.
